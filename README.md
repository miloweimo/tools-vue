# tools-vue

## 项目说明

vue开发的网页工具，使用GitHub page部署

## 安装使用

下载仓库`utilities`和`tools-vue`

```sh
git clone git@github.com:miloweimo/utilities.git
git clone git@github.com:miloweimo/tools-vue.git
```

安装依赖

```sh
cd utilities
npm i
```

启动项目

```sh
npm run serve
```

这时候本地可以预览了！

### 部署脚本

```sh
npm run ghpage
```

网址：https://miloweimo.github.io/utilities/#/
